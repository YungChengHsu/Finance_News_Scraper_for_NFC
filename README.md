# Finance_News_Scraper_for_NFC
Weekly finance news scraper for employees at New Future Capital

## Prerequisites
* [Python](https://www.python.org/downloads/)
* [Anaconda (optional, if you wish not to install many of different packages respectively)](https://docs.anaconda.com/anaconda/install/)

## Dependencies
* [BeautifulSoup4](https://pypi.org/project/bs4/)
* [docx](https://pypi.org/project/docx/)

## Usage
Simply just change the filename you want in the `Weekly_news_scraper.py` file or in the `Weekly_news_scraper.ipynb` if you're using jupyter notebook, and execute the code.   The weekly finance news file will appear in the same folder you place the .py or .ipynb file.
